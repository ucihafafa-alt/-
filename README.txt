Тэнгэрийн Мэлмий PWA суулгах заавар

1. GitHub.com руу орж шинэ repository үүсгэнэ.
2. Энэ folder доторх 4 файлыг бүгдийг upload хийнэ:
   - index.html
   - manifest.json
   - sw.js
   - icon.png
3. Repository → Settings → Pages → Deploy from branch → main / root → Save.
4. Гарсан https://...github.io/... link-ээ Chrome-р нээнэ.
5. Chrome ⋮ → Add to Home screen / Install app.

Анхаарах:
- Файлыг content:// эсвэл file:// гэж нээвэл Add to Home screen гарахгүй байж болно.
- Заавал https:// хаягаар нээхэд апп шиг сууна.
